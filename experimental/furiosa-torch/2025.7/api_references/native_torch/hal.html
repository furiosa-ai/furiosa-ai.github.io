class furiosa.native_torch.hal.Program#

A builder class to build HAL program which runs PeProgram and I/O operations in order.

A syntax for sequential joint, ‘program; program; program’

Parameters:

programs (list[Program])

Returns:

Program

Raises:

ProgramTakenError – If the program has been taken

furiosa.native_torch.hal.copy(src, dst, sequencer=None)#

Copy operation that copy data between DRAM <> host. Does not support host <> host or DRAM <> DRAM. In case that you need copy between DRAM <> DRAM or DRAM <> SRAM, you can use copy_local() method.

Parameters:

  • src (Tensor) – RNGD DRAM tensor | cpu tensor

  • dst (Tensor) – cpu tensor | RNGD DRAM tensor

  • sequencer (Optional[Tuple[Sequencer, Sequencer]]) – src sequencer

Returns:

Program
