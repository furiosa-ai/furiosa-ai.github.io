furiosa.torch.compiler#

furiosa.torch.compiler.compile(aten_fn, inputs, *args, compiler_config=None, summary_to=None, dfg_to=None, edf_to=None, **kwargs)#

compile target aten_fn with inputs and args, it will converted to DFG importable format

Parameters:

  • aten_fn – target aten function

  • inputs – inputs for the aten function

  • args – additional arguments for the aten function

  • summary_to – path to save the dot graph

  • edf_to – path to save the edf graph

  • kwargs – additional keyword arguments for the aten function

Returns:

Edf (furiosa.native_torch.ir.Edf) – Compiled Edf

furiosa.torch.compiler.compile_gm(gm, inputs, compiler_config=None, summary_to=None, dfg_to=None, edf_to=None, cache_dir=None) → Edf#

load or compile the graph module

Parameters:

  • gm (torch.fx.GraphModule) – GraphModule to be compiled

  • inputs (list[torch.Tensor]) – Example inputs for the GraphModule

  • () (compiler_config) – Compiler configuration

  • summary_to – path to save the compilation summary

  • dfg_to – path to save the DFG graph

  • edf_to – path to save the EDF graph

  • cache_dir – compilation result cache directory, it ignores ‘summary_to’, ‘dfg_to’, ‘edf_to’ if set

Returns:

furiosa.native_torch.ir.Edf – Compiled Edf

furiosa.torch.compiler.preprocess_gm(gm, inputs) → GraphModule#

preprocess, which is compiler required, the graph module for the given inputs

Parameters:

  • gm (torch.fx.GraphModule) – GraphModule to be preprocessed

  • inputs (list) – Example inputs for the GraphModule

Returns:

torch.fx.GraphModule – Preprocessed GraphModule
