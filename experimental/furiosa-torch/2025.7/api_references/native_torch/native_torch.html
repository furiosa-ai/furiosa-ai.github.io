furiosa.native_torch.get_device(device_index)#

Get the device handle, first call of this function will initialize the device.

The device is managed in process global manner. You can get the same handle from every function call for the same device_index.

Parameters:

device_index (int) – The index of the device.

Returns:

Device – The device handle of the device_index.

Raises:

PyRuntimeError – If the device is failed to be initialized.

class furiosa.native_torch.Device#

RNGD device handle that can execute HAL Program. The device handle is created when the first call of get_device().

furiosa.native_torch.get_all(device_index)#

Get the allocator handle for the device. It is dual of get_device().

The allocator is managed in process global manner. You can get the same handle from every function call for the same device_index.

Parameters:

device_index (int) – The index of the device.

Returns:

Tuple[Device, Allocator]

Raises:

PyRuntimeError – If the device is failed to be initialized.

class furiosa.native_torch.Allocator#

Allocator handle to manage DRAM, SRAM, and INSTRUCTION allocation

Note

This class is not intended to be used directly. Use torch.to("rngd:?") to allocate memory on the device.

dram_alloc(size)#
