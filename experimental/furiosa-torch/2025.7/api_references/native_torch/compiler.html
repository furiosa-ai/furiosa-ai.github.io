furiosa.native_torch.compiler.compile(gm, compiler_config=None, summary_to=None, dfg_to=None, edf_to=None)#

Compile GraphModule to EDF.

Parameters:

  • gm (torch.fx.GraphModule) – GraphModule to compile

  • compiler_config (Config, optional) – Compiler config to use for this graph.

  • summary_to (str, optional) – The path to save compile summaries.

  • dfg_to (str, optional) – The path to save the DFG file.

  • edf_to (str, optional) – The path to save the EDF file.

Returns:

Edf
