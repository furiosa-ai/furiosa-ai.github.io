class furiosa.native_torch.ir.LoweredAxis#

A LoweredAxis specifies how a multi-dimensional tensor’s axis maps to this linear memory.

It is defined as a tuple of four values: (axis_label, axis_stride, size, buffer_stride)

  • axis_label: A label for this axis. If generated by the compiler (e.g., via AxisProj), its value should be preserved. Otherwise, it can be used for annotation.

  • axis_stride: The stride value specific to this axis unit. If generated by the compiler, its value should be preserved. Otherwise, it should be set to 1.

  • size: The number of elements along this axis.

  • buffer_stride: The number of items (not bytes) to skip in the one-dimensional memory buffer to access the next element along this axis.

LoweredAxis instances are typically not constructed directly by users. Instead, they are generated internally or passed as part of other configurations (e.g., in a Sequencer).

class furiosa.native_torch.ir.Sequencer(axes)#

Defines the memory access order of a Tensor using a sequence of LoweredAxis definitions.

Regardless of its logical shape, every Tensor exists as a one-dimensional array in memory space. The memory access order for each axis of a multi-dimensional tensor is specified by the Sequencer.

A Sequencer is essentially a list (or sequence) of LoweredAxis tuples. Each LoweredAxis tuple provides specific details for one dimension’s memory layout: (axis_label, axis_stride, size, buffer_stride)

For a detailed explanation of each LoweredAxis component, refer to the LoweredAxis documentation.

The following example demonstrates copying the first two elements of each column of a 4x5 tensor to an 8-element DRAM tensor.

Example

import furiosa.torch as _
import torch
from furiosa import native_torch
from furiosa.native_torch import hal, ir
device = torch.device('rngd:0')
cpu_tensor = torch.tensor([0, 1, 1, 0, 0] * 4, dtype=torch.int8)
dram_tensor = torch.zeros([8], dtype=torch.int8, device=device)
program = hal.copy(
    cpu_tensor[1:],
    dram_tensor,
    # src, dst sequencer (`LoweredAxis`)
    sequencer=(
        ir.Sequencer([('srow', 1, 4, 5), ('scol', 1, 2, 1)]),
        ir.Sequencer([('drow', 1, 8, 1)]),
    ),
)
await native_torch.get_device(device.index).execute(program)
expected = torch.tensor([1, 1] * 4, dtype=torch.int8)
assert torch.all(expected == dram_tensor.cpu())
